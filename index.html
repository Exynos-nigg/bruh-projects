What is RAD Kernel?

RAD Kernel is a project oriented towards people who depend on their phone every day. This kernel aims for the maximum stability.

Features & Info

Reliable

All kernel builds are thoroughly tested before Release.

Features Rich

Compiled with Clang 9, O3 optimizations, Boeffla Wakelock Blocker, Disabled CRC Check, Wireguard VPN, Fast, smooth and great for Daily Use!, Need i go on? xD

How to build?

Its very much easy! Clone kernel source and Run bash build.sh! & it you can choose all variables!, and guess what? even the toolchains are included! // Too easy ik!

Compatibility

RAD Kernel per se (and only) supports AOSP ROM(s) and it's mean to be used with it for a better STABLE experience. RAD Kernel will NOT work with any other ROMS except AOSP (i.e no OneUI support). Happy flashing :)

How to flash the kernel?
Step 1

Reboot to your Recovery(TWRP/PBRP) from system or manually by shutting down then pressing Power + Bixby button + Volume UP

Step 2

After booting to recovery, goto wipe section of your recovery and wipe dalvik and dalvik cache (only)

Step 3

Then browse the kernel zip file where you have saved it and flash it as a zip. Reboot and enjoy, Happy Flashing :)
